图绘成都

川大馆藏《全图·成都府图》数字导览

从四川大学图书馆馆藏《清初四川通省山川形胜全图》中的“四川成都府”图出发，将古地图中的桥梁、寺观、祠庙、城址与交通节点转化为可浏览、可点击、可行走的数字导览资源，引导公众观察成都近三百年的城市空间变迁。

Project File

项目档案

《清初四川通省山川形胜全图》以一县一图的方式描绘清初四川山川形胜、府州县治、道路桥梁与人文名胜。本站依据四川大学图书馆馆藏图像资料及“全图项目”整理素材，将古图信息拆解为公众可浏览、可点击、可延展的城市导览单元。

本项目尝试将馆藏古地图从静态图像转化为数字化展示资源，使古籍特藏进入公共阅读、城市教育、研学导览与文化传播场景。

题名
《清初四川通省山川形胜全图》“四川成都府”图
时代
清乾隆初年
形态
绢本设色绘本，一县一图，青绿山水，共 150 幅
馆藏单位
四川大学图书馆
图像价值
记录清初成都府城、桥梁、寺观、祠庙、城址与交通节点，是理解成都城市空间、地名沿革和地方文化记忆的重要图像文献。
资料依据
四川大学图书馆馆藏图像资料、古图局部图像、现地影像、地方文献线索与实地观察记录。
释读原则
点位释读综合古图标注、空间方位、现代地名、道路水系与现地材料进行判断；部分地点仍属初步推定，后续将继续校核。

Citywalk Route

一日步行路线

以“桥、祠、寺、宫、城、驿路”为线索，将古图中的文化地标转译为可行走的城市阅读路线。

路线主题
从桥梁交通、祠庙记忆、佛寺空间、道教宫观、城址变迁到驿路方向，观察清代成都府图中的空间节点如何在现代城市中延续、转化或消失。
推荐时长
半日到一日，可根据实地交通情况分段完成。
适合对象
高校师生、图书馆读者活动、城市文化爱好者、来馆参观与研学团队。
使用方式
可先在线查看古图点位，再按照路线实地观察，最后补充照片、地名线索与现场记录。

    About

    项目说明

    本项目为四川大学图书馆古籍特藏资源数字化展示原型。页面内容基于馆藏《清初四川通省山川形胜全图》相关图像资料整理而成，旨在探索古地图资源在城市阅读、公共教育与数字人文展示中的应用。

    页面中的图片、文字与点位说明仅用于学习、研究和文化传播。部分点位释读仍在校勘中，正式发布前应继续核验馆藏图像授权、点位释读依据、现地影像来源与公众投稿授权。
